# osu-mania-skin.ini-editor

### 简介
这是一个简单的c语言程序，用于为osu!mania模式修改部分skin.ini设置

B站视频介绍：`BV1tsxfzhECY`

### 环境要求
**操作系统**：Linux或Windows7及以上版本  
**内存**：至少512MB  
**存储**：至少10MB  
（不会2025了还有PC机满足不了这个要求吧 =_=）  

**构建要求**  
**Linux**: gcc
**Windows**: mingw-w64  
### 构建
**Linux**

`./build.sh`

**Windows**

`build.bat`
